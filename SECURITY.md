# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 9.x     | :white_check_mark: |
| < 9.0   | :x:                |

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Report vulnerabilities by email to **ggdaltoso@gmail.com** with:

- A description of the vulnerability and its potential impact
- Steps to reproduce
- Any suggested mitigations (optional)

You will receive a response within **5 business days**. If the issue is confirmed, we will release a patch as soon as possible.
